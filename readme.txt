An interface for viewing Formidable Forms responses without upgrading to Pro.

== Description ==

Formidable Forms Pro adds great features to the already-awesome Formidable Forms plugin.
But if the full Pro feature set isn't needed, this plugin lets admins view form
results, normally a Pro feature.

== Installation ==

1. Install the Formidable Forms plugin (http://wordpress.org/extend/plugins/formidable/)
1. Upload the `formidable_results` directory to your site's `/wp-content/plugins` directory
1. Activeate the plugin through the 'Plugins' menu in WordPress
1. Configure!
